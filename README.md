### Run Locally

To run **DestiNex** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/prathikprajapati/DestiNex.git
```

Windows:

```bash
git clone https://github.com/prathikprajapati/DestiNex.git
```
